# HomePro – On Demand Home Services Platform  

🚀 HomePro is a modern MERN Stack-based on-demand home services platform designed to connect users with trusted service providers easily and efficiently. The platform offers a smooth booking experience with AI-powered assistance and location-based service allocation.

---

## 🌟 Features  

- 🤖 AI Chatbot Support  
  Integrated React chatbot to assist users with service queries and navigation.

- 📍 GPS-Based Service Allocation  
  Smart location-based service matching for faster and more efficient bookings.

- 📱 Responsive User Interface  
  Fully responsive and user-friendly design built with React.js and Tailwind CSS.

- 🔐 Authentication System  
  Secure login and user management functionality.

- 🛠️ Service Booking Management  
  Easy booking workflow with streamlined service selection and scheduling.

- ⚡ Modern MERN Architecture  
  Built using MongoDB, Express.js, React.js, and Node.js.
